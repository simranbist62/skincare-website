# 🧴 Skincare Website

A dynamic and responsive skincare e-commerce website built using HTML, CSS, and JavaScript.  
This project includes product filtering and a fully interactive shopping cart system.

---

## ✨ Features
- 🛍️ Product listing with modern UI design
- 🎯 Category and brand-based filtering
- 🛒 Fully functional shopping cart
  - Add to cart
  - Remove items
  - Update quantity
- 💾 Local storage support for cart persistence
- 📱 Fully responsive design
---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 🚀 How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Explore the website
